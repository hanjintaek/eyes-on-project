### ✏️ 제안 배경
공장이나 설비시설의 대규모 화재사례를 예방을 위하여 발열 원인을 신속하게 파악하고 비접촉으로 안전한 점검이 가능케 하고 
전기 설비 온도 측정 및 분석을 통해 유지보수에 용이한 서비스가 필요하다고 생각하여 EYES ON을 기획

<br/>

### 🎮 메인 기능
1. AI를 활용한 이상 예측 및 위험 판단 기능
2. 설비의 위치 및 상태 시각화
3. 이상 발생 시, 즉각적인 알림 서비스 제공
   
<br/>
